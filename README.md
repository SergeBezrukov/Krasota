# Krasota

Приложение для салона красоты. Написано на C# с использованием WPF.

## Что внутри

В программе есть несколько окон для работы:

- **Клиенты** — список клиентов, добавление и редактирование
- **Услуги** — что салон предлагает, типы услуг
- **Мастера** — кто работает в салоне
- **Записи** — кто к кому и когда записан
- **Ежедневное** — скорее всего, расписание или план на день
- **Подключение к БД** — окно для настройки базы данных

## Как запустить

1. Скачать проект или склонировать:
2. Открыть файл `Krasota.sln` в Visual Studio
3. Нажать F5 или кнопку "Запуск"

## Файлы

- `*.xaml` — как выглядит окно
- `*.xaml.cs` — код, который за этим окном стоит
- `App.xaml` — запуск программы
- Картинки — `images (1).jpg` и `Без названия.png` (скорее всего, иконки или фон)

## Что нужно для работы

- Windows
- Visual Studio (любая новая версия)
- База данных (какая именно — надо смотреть в коде)

Если будут вопросы — разбирайтесь в коде, там все просто.
